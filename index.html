My Projects

  • Student Grade Tracking System

    This project was developed by a team of students to help teachers keep track of students' grades and performance. The system allows teachers to input grades, generate reports, and communicate with students. The user-friendly interface and the ability to access grades from anywhere make it an indispensable tool for teachers.

  • Campus Event Management System

    This project was developed by a group of students as a solution for managing events on campus. The system allows students and staff to create, promote, and manage events. It also has features for scheduling and organizing events, as well as sending out reminders to attendees. The system streamlines event management, saving time and resources for the student organizations and the university.

  • Online Food Delivery Platform

    This project was created by a group of students to make ordering food online easier and more convenient. The platform connects customers with local restaurants, allowing them to order food for delivery or pickup. It includes features such as menu management, real-time order tracking, and customer reviews. The platform's user-friendly interface and fast delivery options have made it a popular choice for customers.
